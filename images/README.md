# Images Folder Structure

Place your images here with descriptive names.

## Recommended Organization:

```
images/
├── icons/          (for tab header icons, button icons)
├── backgrounds/    (for background textures, patterns)
├── photos/         (for venue photos, activity images)
└── logos/          (for any logos or branding)
```

## Naming Convention:
- Use lowercase with hyphens: `lobster-icon.svg`, `portland-lighthouse.jpg`
- Be descriptive: `friday-tab-icon.png`, `paper-texture-bg.png`
- Include dimensions if multiple sizes: `logo-small.png`, `logo-large.png`

## File Size Recommendations:
- Icons/SVGs: Keep under 50KB
- Photos: Optimize to 100-300KB each
- Background patterns: Keep under 200KB
